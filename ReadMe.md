Flight Price Aggregator (Python + MySQL)

This project simulates a flight price aggregation system inspired by Skyscanner.
It aggregates prices from multiple platforms, stores them in MySQL, analyzes
optimal prices, and visualizes comparisons using Python.

Tech Stack:
- Python
- MySQL
- Pandas
- Matplotlib

Features:
- 50 cities
- 10 airline operators
- Multiple booking platforms
- Price optimization
- Data visualization

How to Run:
1. Run schema.sql in MySQL
2. Run populate_data.py
3. Run main.py
